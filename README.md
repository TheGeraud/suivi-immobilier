# 🏠 Suivi Achat Immobilier

Application web de suivi complet d'un projet d'achat immobilier — 
de la première visite jusqu'à l'acquisition du bien.

## 🎯 Fonctionnalités

| Section | Description |
|---|---|
| 🏠 **Tableau de bord** | KPIs synthétiques et vue d'ensemble du projet |
| 🏡 **Biens** | Suivi des biens visités et leur statut |
| 🏦 **Banques** | Comparatif des établissements contactés |
| 💳 **Crédit** | Simulation et suivi des offres de prêt |
| 📊 **Analyse** | Graphiques et comparaison des biens |
| 📋 **Documents** | Suivi des documents à fournir |
| 📅 **Calendrier** | Planning des étapes clés |
| 👥 **Contacts** | Carnet d'adresses (agents, notaires, courtiers) |
| 🏢 **Copropriété** | Informations sur la copropriété |

## 🛠️ Stack technique

- **HTML5 / CSS3 / JavaScript** — application 100% front-end
- **Chart.js** — visualisation des données et graphiques
- **Thème clair/sombre** — interface adaptable
- **Responsive** — compatible mobile, tablette et desktop
- **localStorage** — persistance des données en local

## 🚀 Lancer l'application

Aucune installation requise — ouvre simplement `index.html` 
dans ton navigateur :

```bash
# Cloner le repo
git clone https://github.com/TheGeraud/suivi-immobilier.git
cd suivi-immobilier

# Ouvrir dans le navigateur
open index.html   # Mac
start index.html  # Windows
```

## 📁 Structure du projet

suivi-immobilier/
├── index.html    # Structure de l'application
├── style.css     # Styles et thèmes clair/sombre
└── app.js        # Logique métier et gestion des données
